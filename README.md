## 📖 Overview

A simple yet fully functional **Dice Roller App** built using **Jetpack Compose**, Android’s modern UI toolkit.
Developed as part of the **Android Basics with Compose codelab**, this app allows users to roll a digital dice interactively.

This project demonstrates:

* Clean Compose UI structure
* State management in Compose
* Random number generation logic
* UI responsiveness and basic animations

---

## 🧠 Key Learning Outcomes

* Building declarative UIs with **Jetpack Compose**
* Implementing control flow in **Kotlin** (using `when`, `if-else`)
* Managing UI state with **`remember`** and **`mutableStateOf`**
* Enhancing user experience with interactive UI components

---

## 🚀 Features

* **🎲 Roll a Dice:** Tap a button to roll the dice
* **🔄 Random Dice Faces:** Dice images update based on a random number generator
* **🖼️ Real-Time Updates:** Dice face changes instantly on user interaction
* **📱 Responsive Design:** Supports various device sizes with adaptive UI
